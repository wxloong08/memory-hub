Note: Icon files (icon16.png, icon48.png, icon128.png) need to be created.

You can:
1. Use an online icon generator (e.g., favicon.io)
2. Create a simple brain emoji icon
3. Use any 🧠 emoji as PNG in different sizes

For now, the extension will work without icons (Chrome will show a default icon).

To add icons later:
- Create icon16.png (16x16 pixels)
- Create icon48.png (48x48 pixels)  
- Create icon128.png (128x128 pixels)

Recommended: Use a brain emoji or memory-related icon in purple/blue colors.
